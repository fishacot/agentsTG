# Agent task intake

1. Открой **`docs/TASK_TEMPLATE.md`** и используй его как **единственную** целевую структуру.
2. Из последнего сообщения человека извлеки или восстанови поля: цель, scope / out-of-scope, acceptance (чекбоксы), «готово», риски.
3. Выведи **заполненный** шаблон в Markdown. Явно перечисли **что неясно** или пусто — задай вопросы **до** предложения правок в файлах (если только человек не сказал «сразу правь»).
4. Не меняй файлы репозитория на этом шаге, если человек не просил обратного.
