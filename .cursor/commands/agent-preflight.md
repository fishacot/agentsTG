# Agent preflight

Канон: `CursoRules/AGENTS.md`.

Перед правками (опционально **agent-task-intake**):

1. Какие файлы тронешь; что **не** трогаешь (в т.ч. не меняй `CursoRules/` без явной задачи).
2. Будешь вести **`docs/implementation-notes.md`** в корне workspace.
3. Verify: `docs/PROJECT_VERIFICATION.md` (или спроси команды стека).
4. Не предлагай CLI **claw**, если не просили.
5. Минимальный diff.
