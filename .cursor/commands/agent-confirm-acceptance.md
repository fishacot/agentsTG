# Agent confirm acceptance

1. Кратко перечисли **критерии приёмки** текущей задачи (из чата или из шаблона `docs/TASK_TEMPLATE.md`).
2. Спроси человека одним блоком: **«Подтвердите пункты или исправьте формулировки»** (да/нет по каждому пункту или правка текста).
3. Для **рискованных** действий (удаление, переименование публичных путей, mass replace, CI/секреты) добавь отдельную строку: нужно явное **«да, выполняем риск N»**.
4. Пока нет подтверждения — не выполняй рискованные операции; можно готовить план и diff только для безопасной части, если это согласовано.
