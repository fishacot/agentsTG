# AI Agents Telegram Assistant - Системный промпт и архитектура

## 🎯 Цель проекта

Создать интеллектуальную мультиагентную систему в Telegram для автоматизации личной и профессиональной жизни:
- Оркестрация задач между специализированными агентами.
- Управление временем, задачами и заметками в Obsidian.
- Прикладной ресерч: репозитории, best practices, конкуренты, технические решения.
- Исследования в области безопасности ИИ и помощь в кодинге.
- Бизнес-консалтинг и управление проектами.
- Маркетинг и исследование рынка.
- Групповой чат агентов для решения комплексных задач.

## 🏗️ Архитектура системы

### Технологический стек

**Backend:**
- Python 3.11+
- aiogram 3.x (Telegram Bot API)
- LangGraph (Оркестрация агентов)
- SQLAlchemy 2.0 (ORM)
- Alembic (миграции БД)

**База данных:**
- PostgreSQL (основное хранилище)
- Redis (кэш, сессии, Mem0)

**AI модель:**
- Qwen2.5-72B-Instruct через HuggingFace Inference API
- Fallback: OpenRouter API

**Хостинг:**
- Render.com (бесплатный tier)
- Альтернатива: VPS

**Дополнительно:**
- Mem0 (Долговременная память)
- E2B Sandbox (Безопасное выполнение кода)

## 🤖 Описание агентов

### 1. Оркестратор (Orchestrator)
**Роль:** Главный дирижёр, распределяет задачи, следит за контекстом и организует взаимодействие.

**Возможности:**
- Принимает запросы, определяет нужного специалиста.
- Следит за очередностью и приоритетами.
- Организует групповой чат агентов для сложных задач.
- Хранит общую картину проектов пользователя через Mem0.

**Промпт:**
```
Ты - Оркестратор мультиагентной системы. Твоя задача - быть супервизором для команды агентов.
Анализируй запрос пользователя и решай, кому из специалистов его передать.
Если задача требует участия нескольких агентов, организуй их взаимодействие в формате группового чата.
Всегда отвечай на русском языке.
```

### 2. Личный помощник (Personal Assistant)
**Роль:** Управление временем, задачами и информацией.

**Возможности:**
- Интеграция с Obsidian (создание заметок через Markdown/Git).
- Управление Google Calendar.
- Ведение ToDo списков и простого личного бюджета.

### 3. Ресерчер (Research & Intel)
**Роль:** Прикладной исследователь: репозитории, best practices, конкуренты, архитектуры.

**Возможности:**
- Поиск и сравнение технических решений (GitHub, документация, статьи).
- Сбор выжимок по рынку и конкурентам.
- Подготовка списков ссылок и кратких планов действий.

### 4. Агент по безопасности и ИИ (Security & AI Researcher)
**Роль:** Исследование уязвимостей LLM и помощь в разработке.

**Возможности:**
- Поиск CVE, анализ безопасности кода.
- Помощь в отладке через интеграцию с кодом (E2B Sandbox).

### 5. Бизнес-консультант (Business & Project Manager)
**Роль:** Помощь в ведении бизнеса и проектов.

**Возможности:**
- Планирование проектов, расчет бюджетов и рисков.
- Интеграция с Notion/CRM.

### 6. Маркетолог (Marketing & Growth)
**Роль:** Продвижение и исследование рынка.

**Возможности:**
- Написание текстов для соцсетей, анализ трендов.
- Оценка эффективности рекламных кампаний.
