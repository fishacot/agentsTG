# Сниппет для любой задачи (вставь в Harvi / Cursor)

Добавь в **начало** промпта:

```
Соблюдай CursoRules/AGENTS.md (если папка есть в репо).
Веди docs/implementation-notes.md в корне проекта — обновляй после каждой части работы
(решения вне SPEC, tradeoffs, проверки). Шаблон: CursoRules/docs/implementation-notes.TEMPLATE.md
```

Полная формулировка (Thariq):

```
Implement <SPEC> and while you do, keep a running docs/implementation-notes.md
with decisions not in the spec, changes, tradeoffs, and anything I should know.
```
