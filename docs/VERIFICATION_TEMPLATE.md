# PROJECT_VERIFICATION — шаблон

Скопируй в **корень вашего проекта** как `docs/PROJECT_VERIFICATION.md` и заполни команды.

Канон процесса: `CursoRules/AGENTS.md`. Агент сверяется с **этим файлом** в workspace.

## Команды (заполните под свой стек)

| Шаг | Команда | Когда обязательно |
|-----|---------|-------------------|
| Тесты | _например `pytest` / `npm test`_ | После правок backend/логики |
| Сборка | _например `npm run build`_ | После правок frontend |
| Линт | _например `ruff check` / `npm run lint`_ | По возможности |
| Другое | | |

## Цикл повторов

До **3** попыток исправить падение. Затем — стоп, запись в `docs/implementation-notes.md`, вопрос человеку.

## Definition of Done (минимум)

- Задача выполнена в scope.
- Применимые команды из таблицы — успех.
- `docs/implementation-notes.md` обновлён.
