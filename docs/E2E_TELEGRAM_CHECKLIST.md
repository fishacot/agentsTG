## 1. W1 — «Привет» (1 мин)

**Кому:** Егор в ЛС  
**Сообщение:** `Привет`  
**Ожидание:** короткий ответ, без длинного плана из 5 шагов.

---

## 2. W6 / W11 D1 — план 2+ шага (3 мин)

**Кому:** Егор в ЛС  
**Сообщение:**

```text
Сделай план: 1) найди кратко что нового в Python 3.13 2) предложи одну идею для pet-проекта
```

**Ожидание:**

- Одно сообщение с **План:** и кнопкой **Отменить**
- Статус **обновляется в том же сообщении** (Шаг 1/2, 2/2), не десяток спама
- В конце — итог отдельным сообщением или wake

---

## 3. W11 D5 — отмена плана (2 мин)

Повторите запрос с планом, нажмите **Отменить** на первом шаге.  
**Ожидание:** «отменён» / нет новых шагов.

---

## 4. W11 D6 — confirm + replay (3 мин)

**Кому:** Эльза в ЛС  

1. Создайте проект (если нет): `Создай проект тест E2E`  
2. Закройте: `Закрой проект тест E2E` или `Пометь проект тест E2E как завершённый`

**Ожидание:**

- Inline-кнопки **Да** / **Нет**
- Без **Да** — проект **не** done
- После **Да** — подтверждение и выполнение (replay)
